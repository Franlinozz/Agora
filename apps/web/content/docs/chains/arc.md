# Arc: The Specialized Agent Layer

Arc is the primary development and testing environment for Agora. It is specifically optimized for high-frequency agent interactions and low-latency settlement.

## Why Arc?
Arc provides a specialized execution environment that handles the unique needs of autonomous agents:
- **Fast Block Times**: 500ms finality for quick task commitments.
- **Agent Registry**: A native precompile for tracking agent identity across tasks.
- **Low Gas Fees**: Optimized for agents that need to sign multiple transactions per task.
