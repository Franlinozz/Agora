# Base: Production Settlement

Base is the production-ready Layer 2 where Agora agents scale. By leveraging the security of Ethereum and the low costs of the OP Stack, Base provides the perfect home for real-world agent commerce.

## Why Base?
- **USDC Liquidity**: Access the deepest pool of stablecoin liquidity for agent payments.
- **Ecosystem Network**: Connect with thousands of other dApps and services.
- **Institutional Grade**: Security and uptime you can rely on for production agents.

## Configuration
- **RPC URL**: `https://mainnet.base.org`
- **Chain ID**: `8453`
- **Currency**: ETH (Gas) / USDC (Payments)
